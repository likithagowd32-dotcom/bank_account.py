# bank_account.py
# Project Overview
This project is a simple **Bank Account Management System** developed using **Object-Oriented Programming (OOP)** concepts in Python.  
It simulates real-time banking operations such as deposit, withdrawal, interest calculation, and account management.

This project is created as part of **TASK 10: Object-Oriented Programming (OOP)**

# Tools & Technologies
- Python
- VS Code

# Objectives
- Understand Object-Oriented Programming concepts
- Implement real-world banking operations
- Learn code structuring using classes and objects
- Practice inheritance, encapsulation, and method overriding
#Features
- Create bank accounts
- Deposit money
- Withdraw money
- Savings account with interest
- Current account with overdraft limit
- Display account details
- Multiple account objects

# How to Run the Project
1. Install Python on your system
2. Open VS Code
3. Create a file named `bank_account.py`
4. Copy and paste the project code
5. Open terminal and run:

Account details displayed
